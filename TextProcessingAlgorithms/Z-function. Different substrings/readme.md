##Число различных подстрок

Имя входного файла: input.txt\
Имя выходного файла: output.txt\
Ограничение по времени: 2 с\
Ограничение по памяти: 256 МБ\
Найти количество различных подстрок строки 
s, используя z-функцию.

**Входные данные**\
Первая и единственная строка входного файла содержит строку 
s
 (
1
≤
|
s
|
≤
10
000
). Строка состоит только из строчных латинских букв.

**Выходные данные**\
Выведите одно число — количество различных подстрок.

#### Примеры
1. входные данные\
aaa\
выходные данные\
3

2. входные данные\
abacaba\
выходные данные\
21
3. входные данные\
bbbbaaabababba\
выходные данные\
79
4. входные данные\
bbaaaabaaaabbaab\
выходные данные\
96
5. входные данные\
mbwprelbayy\
выходные данные\
64
